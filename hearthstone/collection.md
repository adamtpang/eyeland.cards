# Collection — Rafaamlock card pool

Every card that appears in current competitive Rafaam Warlock ("Rafaamlock")
lists, verified against hearthstone-oracle on **2026-07-11** (week 1 of Escape
from Violet Hold, patch 36.0).

**How to use:** check `[x]` when you own **all needed copies** (2× lines need
the pair). Own 1 of 2? Write `1/2` next to it. Claude treats unchecked = not
owned and will flag dust + offer substitutes before proposing those cards.

**Dust per craft:** Common 40 · Rare 100 · Epic 400 · Legendary 1600.
Core-set cards are free (auto-granted) — no checkbox needed, listed for
completeness.

**Build tags:** `both` = in big and cycle builds · `big` = XL/Godfrey big
build · `cycle` = cycle/removal build · `tech` = fringe/flex.

- [ ] 1× **Timethief Rafaam** — Legendary · Across the Timeways · **1600** · both

| ------ | ---- |
| Both-builds core only (Timethief, Godfrey, Elise + both-tag cards) | **6,360** |
| Full cycle build | **7,840** |

**Craft order if starting thin:** Timethief → Godfrey → the cheap `both`
commons/rares (RAFAAM LADDER!!, Shadow Rounds, Caged Cranium, Defias Smuggler,
The Unseen Atlas, Cursed Catacombs) → Elise → then pick ONE build to finish
(cycle is 3,200 dust cheaper; big is the higher-sample consensus).
